xPC Target    

SCRAMNet GT PCI

The SCRAMNet GT PCI board is a real-time reflective PCI memory board. It can also generate/broadcast interrupts. xPC Target uses this board as part of the shared memory network that you can use to exchange data between computer nodes.

xPC Target supports this board with these driver blocks:

  • SCGT Init
  • SCGT Read
  • SCGT Write
  • SCGT Get State
  • SCGT Set State
  • SCGT Send Interrupt

IO421-3 / IO422-3 LVDT/RVDT connection methods

When simulating an LVDT/RVDT with the IO421-3 / IO422-3 sub-module the following connection methods can be used.

 

4-wire LVDT

 

The default configuration is the 4-wire LVDT/RVDT:

 

•If the position of the rod reaches the top in the below diagram (at Va), then Va outputs a sine wave with the same amplitude and frequency as the reference voltage. The Vb output is zero.

•If the position of the rod reaches the bottom, then Vb outputs a sine wave with the same amplitude and frequency as the reference voltage. The Va output is zero.

•If the rod is in the center, Va and Vb are 180 degrees out of phase and each output has half of the amplitude of the reference voltage.

IO331 > Driver configuration > EnDat

Input and output ports

Input and output ports

This EnDat driver block has no inputs, and up to four output ports.

 

 

Output port

Description

Position

The encoder position value.

Position Width

The number of bits for the position value representation.
Used for multi-turn encoder (also visible if "Automatic detection of encoder data width" is checked)

Revolution

The encoder revolution value.
Used for multi-turn encoder (also visible if "Automatic detection of encoder data width" is checked)

ErrorBit

The error bitfield, decoded as follows:
- Bit 0: F-Typ III Error 1F
- Bit 1: F-Typ III Error 0F
- Bit 2: F-Typ II  Error
- Bit 3: F-Typ I   Error
- Bit 4: CRC Error
- Bit 5: Error bit 2 of EnDat protocol (Alarm2)
- Bit 6: Error bit 1 of EnDat protocol (Alarm1)
- Bit 7: FPGA configuration error

Dialog fields

Tab: Module setup

 

 

Parameter

Description

FPGA module identifier

As it's possible to have more than one reconfigurable FPGA module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one FPGA module then leave the default value (1).

FPGA based IO module

Select the name of the FPGA-based I/O module for which you are configuring setup parameters.

PCI Slot (-1: autosearch)

If only one module of this type is present in the target machine, leave as default ( '-1') to automatically locate the board. If more than one module of this type is present, enter the bus number and the PCI slot number of the board associated with this driver block using the format '[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

 

 

Tab: Bitstream

 

 

Parameter

Description

FPGA bitstream

This parameter defines the FPGA bitstream of the set of FPGA code modules to download to the FPGA chip of your reconfigurable FPGA module.

The FPGA bitstream is downloaded to the FPGA chip at the time your Simulink model is built and sent to the target machine.

If you are using the default bitstream please refer to the technical specifications manual of your FPGA-based I/O module to find further information including the bitstream name and the I/O pin mapping.

If you are using a custom implementation, please refer to the manual providing further information about the name of your custom bitstream and the custom I/O pin mapping.

Tab: Conditioning plug-in, Conditioning module: IO331_21

 

 

Parameter

Description

port 0 - 6

The plug-in “IO331_21” supports 7 ports. Each of this port have 8 digital I/O-lines. The I/O voltage (3.3V or 5.0V) and the pull resistor (pull-up, pull-down, floating) for each port are selectable. All of the ports are completely independent. But the direction of each port has to be the same for all 8 I/O-lines. The direction of the I/O-lines is implicitly given by the count of channels of the digital I/O blocks which are used in the current model.

pull resistor

Select the pull resistor of the 8 digital I/O-lines of the selected port. Floating pull resistor are only allowed for ports which are used as output.

I/O voltage

Select the I/O voltage of the 8 digital I/O-lines. Since the I/O voltage has to be the same as the pull-up voltage, this pop-up will disappear if a pull-up resistor is selected in the pull resistor pop-up.

Center-aligned

Here is a timing diagram for a center-aligned (symmetrical) signal:

 

Key:

dashed line - counter

red line - PWM output

 

 

Here the 'PWM pattern' parameter is set to PWM is symmetric and centered on half the period (see the PWM Generation tab in the module dialog).

 

In this configuration only the AOn value is required. AOff is automatically calculated by the driver so that the PWM output is symmetrical about the midpoint (half the period).

 

If AOn is set to a value greater then the midpoint then an inverted signal will be generated.

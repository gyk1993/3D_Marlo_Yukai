IO331 > Driver configuration > PWM generation driver block

Input and output ports

Input and output ports

This driver block has up to 6 input ports depending on which input ports are enabled in the dialog fields of the PWM generation driver block, and contains no output ports. The possible input ports are:

 

Input port

Description

A

The A On compare value.

This port is only active when the PWM pattern is symmetric (centered on half the period). A Off will be computed by the driver to ensure the symmetric PWM generation.

B

The B On compare value.

This port is only active when the PWM pattern is symmetric (centered on half the period). B Off will be computed by the driver to ensure the symmetric PWM generation.

A On

The A On compare value.

This port is only active when the PWM pattern is asymmetric.

A Off

The A Off compare value.

This port is only active when the PWM pattern is asymmetric.

B On

The B On compare value.

This port is only active when the PWM pattern is asymmetric.

B Off

The B Off compare value.

This port is only active when the PWM pattern is asymmetric.

 

All input ports are of data type double. The widths of the input ports have to be equal to the number of PWM channels as defined in dialog field parameter “channel vector” of this driver block. Each element can be in the range from 1 to 2^32-1.

 

Note: It is the responsibility of the user of the block to make sure that the A-compare, B-compare, and Period-compare values are in proper relation to each other.

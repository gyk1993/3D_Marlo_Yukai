IO331

The IO331 is a multi-function FPGA-based I/O module with Xilinx Spartan 6 chip with 147k logic cells and 32 LVDS or 64 2.5V digital I/O lines.

 

 

This I/O module provides up to 64 rear-access, single-ended LVCMOS 2.5V digital I/O lines, or up 32 rear-access, differential LVDS I/O lines. The two types can be combined.

 

The IO331 can be optionally extended with one of the following plug-ins:

 

Front plug-in modules for the IO331

IO331-1

64 x digital 3.3V/5V TTL I/O LVTTL front I/O lines

IO331-1-LV

64 x digital 2.5V LVTTL front I/O lines

IO331-2

30 x RS485 (RS422 compliant) front I/O lines

IO331-3

16 x TTL (5V) and 22 RS485 (RS422 compliant) front I/O lines

IO331-4

30 x LVDS I/O lines

IO331-5

2 x 16-bit 105 MHz differential simultaneous analog inputs

IO331-6

16 x 16-bit 500kHz ADs, 8 x 16-bit DAs with 10us settling time, 16 x digital TTL I/O lines

IO331-7

16 x 16-bit analog outputs with 2us settling time, +/- 10V

 

Rear plug-in modules for the IO331

IO331-21

Converts the rear I/O lines from LVCMOS to 3.3V/5V TTL and makes them accessible from the front of the enclosure.

 

Speedgoat implements (as standard, and for no extra cost) the following FPGA Code Modules (one FPGA Code Module = one channel) on the IO331 or the IO331-X plug-in I/O lines:

 

•3 x PWM, 3 x CAP , 3 x QAD, and 1 x Interrupt.

 

The remaining I/O lines can be used as general purpose digital inputs or outputs. Different FPGA Code Module implementations can be provided by Speedgoat at any time by e-mail. Please contact us for more information.

 

The following standard FPGA Code Modules are available for the IO331:

 

Pulse Width Modulation

PWM

Pulse generation for symmetric, asymmetric, single-phase, multiphase, and dead-band compensation and frequency-modulated pulse trains

CAP

Capture channels to measure pulse and period durations and PWM duty cycles

 

Encoder Measurement and Simulation

QAD

Quadrature decoding of incremental encoder sensors to determine rotation direction, position, and encoder index

QAE

Quadrature encoding module, simulating incremental encoder sensors

SSI Master

Decoding of absolute encoder sensors

SSI Slave

Emulation of absolute encoders

EnDat

EnDat protocol implementation to communicate with encoder sensors from Heidenhain supporting EnDat 2.2

BiSS

BiSS protocol implementation to communicate with encoders from Renishaw

 

Serial Protocols

SPI

Implements SPI Master or SPI Slave protocol functionality

I2C

Implements I2C Master or I2C Slave protocol functionality

 

Synchonization

INT

Interrupt to pace (synchronize) the entire model or a function-call triggered subsystem using an external source signal. The source signal can come from an FPGA-based I/O module (e.g. from a PWM Trigger output or to replace the CPU time with a interrupt pulse train), from another I/O module, or from external equipment.

SYN

Provides functionality to synchronize various I/O modules installed in the same or in different target machines

 

 

Depending on your requirements Speedgoat can implement a different channel count for the FPGA code modules at any time, or integrate additional functionality such as SPI or I2C protocol support.

 

The IO331 I/O module can also be used to execute your Simulink design using automatically generated HDL Code using HDL Coder from MathWorks (see "Simulink Real-Time model preparation for HDL real-time execution using the IO331"), or to execute your own HDL Code created using Speedgoat FPGA Engineering Kits.

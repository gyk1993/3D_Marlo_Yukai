IO511 > Driver_configuration > Enable TX interrupts block

Dialog fields

Dialog fields

 

Parameter

Description

IO511 module identifier

As it's possible to have more than one module of this type in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one module of this type then leave the default value (1).

Channel

This parameter indicates which serial controller will be allowed to generate the interrupt.

Sample time

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

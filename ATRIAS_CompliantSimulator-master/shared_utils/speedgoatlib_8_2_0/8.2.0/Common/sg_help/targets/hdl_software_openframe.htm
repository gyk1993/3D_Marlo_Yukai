Xilinx HDL software

Xilinx HDL software

If you use Speedgoat IO3xx reconfigurable FPGA-based I/O modules together with Simulink HDL Coder or Speedgoat FPGA Engineering Kits, the following Xilinx HDL software must be installed:

 

I/O Module

Software

Comments

IO301-IO304

Xilinx ISE WebPack 10.1

The WebPack is available at no charge. Supports Windows XP only!

IO311-IO314

Xilinx ISE Design Suite 10.1

(newer versions do not support the IO311)

If a Xilinx ISE license is not available, Xilinx Vivado Design Suite (including Xilinx ISE Design Suite V14.7) must be purchased. Instructions on how to receive license files for V10.1 are provided below.

 

Xilinx ISE Design Suite 10.1 does not officially support Windows 7, but the 32-bit version of V10.1 is stable on this operating system.

IO321

Xilinx ISE Design Suite 10.1 or newer

If a Xilinx ISE license is not available, Xilinx Vivado Design Suite (including Xilinx ISE Design Suite V14.7) must be purchased.

IO331

Xilinx ISE Design Suite 14.x, provided together with Xilinx Vivado Design Suite.

If a current Xilinx ISE license is not available, Xilinx Vivado Design Suite (including Xilinx ISE Design Suite V14.7) must be purchased. The same license file can be used for both products.

IO341

Xilinx Vivado Design Suite

 

 

You do not require Xilinx software if you are using IO3xx FPGA-based I/O modules together with custom bitstreams provided to you by Speedgoat, implementing your required channel count and type(s) of FPGA Code Modules. In this case, the IO3xx I/O module will function like any other fixed-functionality I/O module.

 

How to get license files for previous versions of Xilinx ISE (required for IO301-IO314):

 

Visit: www.xilinx.com/getlicense/

-> [Log in with your User ID and your Password]

-> [Personal Profile / Click Next]

-> [Product Licensing / Go to „Legacy Licensing“ Tab]

-> Obtain a Registration ID for 10.4 thru 10.1

-> [10.x and 9.x Software Registration ID Request / Please click Next]

-> [Fill in the survey]

-> Next

IO92X - IO97X > Programmable Resistor Control block

Input and output ports

Input and output ports

Input port

Description

Rn

(n is the resistor channel number)

The requested resistance value.

 

The number of these ports is dependent on the channel count, which is specified by the 'Channels vector' parameter.

 

Output port

Description

Rn

(n is the resistor channel number)

The value that the driver is trying to assign to the resistor.

This value may differ slightly from the input value and the actual value of the resistor.

 

The number of these ports is dependent on the channel count, which is specified by the 'Channels vector' parameter.

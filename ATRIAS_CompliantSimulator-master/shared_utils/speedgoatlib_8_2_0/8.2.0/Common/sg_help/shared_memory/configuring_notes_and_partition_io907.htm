Configuring notes and partitions

Some configuration items are required, others are optional. The following M-file shows the minimum parameters to set for Master/Slave communication.

%% Sample time

Ts = 0.001;

 

%% Partitions definition

Partitions(1).Size = '1';

Partitions(1).Type = 'uint8';

 

Partitions(2).Size = '2';

Partitions(2).Type = 'double';

 

Partitions(3).Size = '1';

Partitions(3).Type = 'double';

 

Partitions(4).Size = '1';

Partitions(4).Type = 'uint8';

 

Partitions(5).Size = '2';

Partitions(5).Type = 'double';

 

Partitions(6).Size = '1';

Partitions(6).Type = 'double';

 

Partitions = completepartitionstruct(Partitions,'5565');

 

%% Nodes definition

 

MasterNode.Interface.NodeID = '1';

MasterNode.Partitions = Partitions;

MasterNode = completenodestruct(MasterNode,'5565');

 

SlaveNode.Interface.NodeID = '2';

SlaveNode.Interface.Interrupts.PendingInt1='on';

SlaveNode.Partitions = Partitions;

SlaveNode = completenodestruct(SlaveNode,'5565');

 

It’s important to call this initialization before opening the models. The configuration file is config_IO907.m and it’s called in the PreLoadFcn and InitFcn callbacks of the models.

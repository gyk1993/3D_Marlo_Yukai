Input and output ports

Input and output ports

This driver block has n scalar input ports of type double where n is the number of selected analog output channels. Each port (signal) accepts the D/A value in Volts to be output.

 

 

Input port

Description

X

 

(X is the D/A channel number)

The value to be output from D/A channel X in Volts.

 

Type double.

IO642 > Profibus DP Slave Network Configuration

SYCON.net Configuration Tool

SYCON.net Configuration Tool

Currently the only way to configure the IO642 Profibus DP Slave is by using the SYCON.net tool from Hilscher.

The software is available on the USB stick delivered with your real-time target machine or it can be downloaded from our FTP server right here.

 

1.Open SYCON.net and locate the “CIFX DP/DPM” Profibus DPV1 Master. Drag it to the blue line in the main panel. This creates a magenta colored Profibus line.

 

 

 

2.Locate the “CIFX DP/DPS” Profibus DPV1 Slave from the "Gateway / Stand-Alone-Slave" section and drag it to the Profibus line in the main panel.

 

 

3.Open the Configuration window from the slave's context menu.
 

 

4.Under Configuration/Modules, define the I/O data modules to be supported by the slave by appending them to the "Configured Modules" list.

 

 

Note: The Input modules are actually the outputs of the slave and vice versa. It is named after the master's point of view.

 

5.The slave address is set to '2'. It is possible to change this in the station table of the master device's configuration.
 
6.Apply the configuration and close the windows. Right click on the Slave icon and select Additional Functions, Export, DBM/nxd

 

 

7.Select a name for the file, e.g.: my_slave.nxd. Save the file in your current MATLAB workspace.

 

8.In MATLAB, locate the file and use the following script to load the configuration and the firmware to the target machine

 

IO642                 = sg_IO75X_struct();   % Define default structure

 

IO642.protocol        = 'IO642';             % Protocol Speedgoat ID

IO642.cardid          = '1';                 % Card ID

IO642.configfiles     = '1';                 % No. of configuration files

IO642.configfile1     = 'my_slave.nxd';       % Configuration filename

 

sg_IO75X_loadconfig(IO642);  

 

9.Configure the driver blocks according to the configuration defined in SYCON.net by specifying the corresponding amount of bytes in the "PBS Receive" and the "PBS Send" block.

 

10.When you download the model, the firmware and configuration files are loaded to the IO642 card.

Communication between two IO710/IO713 Ethernet ports available in the same Real-time target machine

Let's assume you have two IO710/IO713 I/O modules in the same target machine or an IO710/IO713 I/O module with multiple Ethernet ports. Then you can do a simple loopback test using a single Simulink model.

 

To do so first combine the two models from the previous example into one new model:

 

 

You have to delete one "Buffer Mngmt" block and to adapt the ID value to “2” in one of the "Ethernet Init" and "Ethernet Rx" block.

 

You also need to identify the PCI bus and slot numbers of both IO710/IO713 Ethernet ports and update the two "Ethernet Init" blocks.

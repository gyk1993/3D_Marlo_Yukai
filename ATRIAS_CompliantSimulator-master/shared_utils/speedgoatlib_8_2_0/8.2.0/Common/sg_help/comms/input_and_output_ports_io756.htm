IO756 > Driver configuration > Network status driver block

Input and output ports

Input and output ports

Input port

Description

Network Status

Allows the Network Status to be set.

Possible values:

•0: Network communication disabled

•1: Network communication enabled

Type uint8.

This port is enabled or disabled by the 'Set Status Port' parameter in the dialog box of this block.

 

 

Output port

Description

Network Status

Allows the Network Status to be viewed.

Possible values:

•0: Network communication disabled

•1: Network communication enabled

Type uint8.

This port is enabled or disabled by the 'Get Status Port' parameter in the dialog box of this block.

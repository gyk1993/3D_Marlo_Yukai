Speedgoat Real-Time UDP Libraries

Speedgoat Real-Time UDP Libraries

Important Note

The Simulink Real-Time UDP library from MathWorks does not allow configuring multiple Ethernet ports, as the “Network Configuration” block has no option to select more than one controller.

 

The IO713 I/O module from Speedgoat provides four Ethernet ports. If you want to use more than one Ethernet port at a time for real-time UDP, then use the extended Speedgoat driver library:

 

>> speedgoatlib/IO71X/Real-Time UDP

 

 

 

The usage of the library blocks is the same as with the Simulink Real-Time based real-time UDP blocks. The last demo of this manual uses the Speedgoat library to perform a loopback test between two ports of a single IO710/IO713 I/O module.

Parameter

Description

IO421 / IO422 module identifier

As it's possible to have more than one IO421 / IO422 module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one IO421 / IO422 module then leave the default value (1).

Slot

The carrier board can contain up to three modules. This parameter indicates where your IO421-4 / IO422-4 sub-module is located.

Expected excitation voltage vector

This parameter is used with the 'VLL output level vector' parameter to calculate the gain which is then applied to the reference voltage to produce the output voltage.

Gain = VLL output level vector / Expected excitation voltage vector

Units are Vrms.

VLL output level vector

This parameter defines the output amplitude of the synchro or resolver simulation output.

Units are Vrms.

Ratio CH1/2

The ratio between coarse (channel 1) and fine (channel 2) channels.

Enter the desired ratio, as an integer number, in the D/S Ratio Register corresponding to the pair of channels to be used as a two-speed channel.

By entering a ratio in the D/S Ratio 1/2 register, the fine (channel 2) channel will automatically output a signal proportional to the programmed coarse channel times the ratio programmed.

Example: Single speed = 1; 36:1 = integer 36.

This parameter has a range from 1 to 255.

Output mode

The representation of the output value.

Options are:

•Ratio-metric: the output is calculated as a ratio of the reference voltage

•Fixed: the output value

Rotation mode

The simulation method.

Options are:

•Speed driven: the output is continuously updated depending on the value of the 'rotation rate' input port.

•Start and stop: the output is first set to the angle in the 'start angle' input port, then changes at the speed in the 'rotation rate' input port to the angle in the 'stop angle' input port.

•Angle driven: the output is continuously updated depending on the value of the 'angle' input port.

Output format

Output format

•Synchro

•Resolver

Phase offset vector

The phase of each individual channel may be offset from Reference.

The phase may be adjusted at a resolution of 0.1 deg / Bit.

The range of this parameter is -90 to 90.

Sample time [s]

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

PCI slot (-1: autosearch)

If only one module of this type is present in the target machine, leave as default ( '-1') to automatically locate the board. If more than one module of this type is present, enter the bus number and the PCI slot number of the board associated with this driver block using the format '[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

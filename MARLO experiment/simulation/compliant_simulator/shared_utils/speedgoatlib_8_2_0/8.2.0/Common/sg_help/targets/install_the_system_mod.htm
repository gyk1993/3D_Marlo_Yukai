Target machine installation

Your Modular real-time target machine can be placed on a stable desk or in a rack, by installing the provided rack handles.

 

You can easily change the handles.

 

Monitor

 

You can connect one VGA, DVI or DisplayPort monitor to your target machine, depending on the available connectors and system configuration. The monitor is not mandatory, but recommended for kernel transfer and debugging operations.

 

For the first session with your new target machine, please connect a monitor. The target main screen provides helpful informations:

•Simulink Real-Time version
•Number of CPU cores loaded
•Available memory
•TCP/IP configuration for host-target communication

 

You can also use Target Scopes in the Simulink model to visualize signals and parameters at execution time.

 

If you don't use a monitor during real-time operations, you could run the kernel in text mode, to improve performance.

 

If you do use a monitor, connect it before powering up the system. Make sure that the monitor is well connected and do not unplug it during model execution. This could have an impact on the application stability and in some particular cases, crash the system. If this problem occurs, you must restart the target machine.

 

Keyboard

 

Depending in your system configuration, you can use PS/2 and/or USB keyboards. See the Technical Reference Manual for further information.

 

PS/2 keyboard: enter the BIOS, use DOS commands, use Simulink Real-Time commands

USB keyboard: enter the BIOS, use Simulink Real-Time commands

 

The keyboard is not mandatory for Simulink Real-Time.

 

To activate the command line interface once you have a target application running please type C at the attached keyboard. For a list of commands available in the Simulink Real-Time environment, please refer to the Simulink Real-Time documentation.

 

Power supply

 

Connect the provided power cord to the power supply at the rear of the target machine. For the technical specification of the power supply and voltage input range, please refer to the Technical Reference Manual, specific for your target machine.

 

The power supply has a main switch, near the power cord connection. When this switch is in position ON, the motherboard is powered.

 

Power-on / Power-off

 

To power on the system, use the switch on the front of the target machine (upper-left).

 

The system usually takes a few seconds to load the Simulink Real-Time kernel stored in the target machine main drive. Once the kernel is loaded, you will visualize some kernel information on the target screen.

 

To power-off the system, use the same switch.

TAB: Module setup

 

 

Parameter

Description

FPGA module identifier

As it's possible to have more than one reconfigurable FPGA module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one FPGA module then leave the default value (1).

Channel vector

The channels to use.

Each driver block is for use with a single channel only. To use extra channels please add additional driver blocks to your model.

Sample time

Defines the base sample time at which this driver block gets its sample hit.

It can also be set to -1 for inherited sample time.

PCI Slot (-1: autosearch)

If only one module of this type is present in the target machine, leave

as default ( '-1') to automatically locate the board. If more than one

module of this type is present, enter the bus number and the PCI slot

number of the board associated with this driver block using the format

'[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

 

Tab: BiSS Sniffer configuration

 

 

 

Parameter

Description

Clock time out

The time-out to abort the current communication in case insufficient master clock cycles are received.

Units are in ticks*

Range 0 to 216

 

 

*                For IO30X and IO31X: 1 tick = 1/33Mhz

For IO331: 1 tick = 1/75Mhz

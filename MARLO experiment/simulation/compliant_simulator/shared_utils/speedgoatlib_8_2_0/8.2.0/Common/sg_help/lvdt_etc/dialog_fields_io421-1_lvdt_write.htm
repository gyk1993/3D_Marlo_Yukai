Dialog fields

Dialog fields

Tab: Module setup

 

 

Parameter

Description

IO421 / IO422 module identifier

As it's possible to have more than one IO421/IO422 module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one IO421/IO422 module then leave the default value (1).

Slot

The carrier board can contain up to three modules. This parameter indicates where is your IO421-1 located.

Channels vector

Enter the channels to enable. A correct value is a scalar or a vector where the elements are 1, 2 or 3.

Wiring mode

The wiring method for the connected LVDT/RVDT.

Options are:

•2 wires

•4 wires

 

See section 'IO421-1 LVDT/RVDT connection methods' in the 'Usage notes' of this document for information about the various connection methods.

Sample time

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

PCI slot (-1: autosearch)

If only one module of this type is present in the target machine, leave as default ( '-1') to automatically locate the board. If more than one module of this type is present, enter the bus number and the PCI slot number of the board associated with this driver block using the format '[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

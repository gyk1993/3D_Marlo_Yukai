This example uses another Simulink model for data retrieval, but one could easily replace the host model by any other application capable of receiving UDP messages.

 

Create a new model in Simulink. Open the "slrtlib/UDP" library and drag and drop the "Receive" and "Unpack" blocks. Then add a Scope from the "Simulink/Sinks" library.

 

Go to “Simulation, Configuration Parameters, Solver panel”:

-Simulation time, Stop time: 20
-Solver options, Type: Fixed-step ; Solver: discrete

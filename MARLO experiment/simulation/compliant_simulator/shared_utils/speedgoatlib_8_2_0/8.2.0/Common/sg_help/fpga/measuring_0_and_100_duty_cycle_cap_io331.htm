Measuring 0% and 100% duty cycles

When the duty cycle of the PWM signal is 100% (or 0%) the type of trigger used to start the measurements becomes more important. When a 0 or 100% duty cycle occurs over two or more PWM periods, the positive or negative trigger is missing from the second and successive periods.

 

The diagram below shows a PWM signal with duty cycles 60%, 100%, 100% and then 50%.

The positive trigger is used which works well for the first period (60%). However, for the next three periods there is only one trigger  - there is no positive transition on the signal  between the start of the second period and the start of the fifth period. In this case no separate measurements will be made for periods 3 and 4. The measurements for period 2 will include those of periods 2, 3 and 4.

 

In the following diagram (with the same PWM signal as before), an external trigger is used. This is configured to trigger at the beginning of each PWM cycle. In this case all PWM cycles are correctly measured.

 

 

Alternatively a software trigger could be used. For this example, if the time between the software triggers is the same as the PWM period then the measurement results will be the same as with the external trigger.

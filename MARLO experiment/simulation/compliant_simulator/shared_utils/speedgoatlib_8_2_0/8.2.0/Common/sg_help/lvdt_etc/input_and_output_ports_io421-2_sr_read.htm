IO421, IO422 > Driver configuration > IO421-2 / IO422-2 Synchro or resolver measurement > Synchro/Resolver read driver block

Input and output ports

Input and output ports

 

Input port

Description

Max RPS

The maximum rotational speed.

 

Units are RPS (revolutions per second).

 

This port is enabled or disabled in the 'Input and output ports configuration' tab of this block's dialog box.

The width of this port is dependent of the number of channels (defined by the 'Channels vector' parameter).

 

Output port

Description

Position

The measured position.

 

Units are Radians.

 

This port is enabled or disabled in the 'Input and output ports configuration' tab of this block's dialog box.

The width of this port is dependent of the number of channels (defined by the 'Channels vector' parameter).

Velocity

The measured velocity.

Units are RPS.

This port is enabled or disabled in the 'Input and output ports configuration' tab of this block's dialog box.

The width of this port is dependent of the number of channels (defined by the 'Channels vector' parameter).

Status

A bit field indicating test, signal and reference statuses.

•Bit0: the status of the BIT (built-in tests procedure).

•Bit1: the signal status. If the signal is below 90% of the expected threshold, this flag is high.

•Bit2: the reference status. If the reference is below 90% of the expected threshold this flag is high.

This port is enabled or disabled in the 'Input and output ports configuration' tab of this block's dialog box.

The width of this port is dependent of the number of channels (defined by the 'Channels vector' parameter).

 

 

All ports are vectors. The width of the vector, and the order of the corresponding channels, is dependent on the configuration of the 'Channel vector' parameter.

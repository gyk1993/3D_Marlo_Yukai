Input and output ports

Output port

Description

CAN Port 1 status

Various status information about CAN port 1 packed into a scalar uint16.

The 'Read status CAN port' driver block can be used to unpack this. See the 'Read status CAN port' driver block for details of the available status information.

CAN port 1 bus last

This value is equal to the percentage (0..100) of the total CAN bus load on CAN port 1.

CAN Port 2 status

Various status information about CAN port 2 packed into a scalar uint16.

The 'Read status CAN port' driver block can be used to unpack this. See the 'Read status CAN port' driver block for details of the available status information.

CAN port 2 bus last

This value is equal to the percentage (0..100) of the total CAN bus load on CAN port 2.

CPU last

This value is equal to the percentage (0..100) of the total CPU/firmware load of the IO601 module.

Life Counter

This value counts up and wraps based on an internal IO601 counter. This value can be used to check if the firmware on the IO601 is still running (values change) and therefore as a test to see if the IO601 module is working properly.

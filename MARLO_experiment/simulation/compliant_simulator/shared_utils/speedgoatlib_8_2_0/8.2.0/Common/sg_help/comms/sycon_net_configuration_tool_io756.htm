SYCON.net Configuration tool

SYCON.net Configuration tool

The SYCON.net configuration tool is used to configure the EtherNet/IP Adapter.

 

1.Open SYCON.net and locate the “CIFX RE/EIS” Stand-Alone Slave. Drag and drop the device to the main panel.

 

 

2.Double-click on the icon to open the Configuration window.

 

3.Navigate to Configuration, General. You can select all available options.

 

 

4.Navigate to Configuration, Electronic Keying. You can select the Keying method.

 

 

5.Navigate to Configuration, Connection. Usually you don’t need to modify these parameters. For use with Rockwell RSLogix controller, keep the default.

 

•Originator to Target: 32-bit run / idle header
•Target to Originator: Connection is pure data and is modeless

 

 

6.Navigate to Configuration, Assembly. Define IN and OUT assembly instances. Maximum length for each instance is 504 bytes. The Instance ID can be modified.

 

 

Note: IN/OUT instances are referenced to the Scanner/Master. The IN assembly instance is the output instance for the Adapter/Slave. As a consequence, the OUT assembly instance defines the input data for the Adapter/Slave.

 

7.Navigate to Configuration, Device Settings. Please keep the default configuration.

 

 

8.Apply the configuration and close the windows. Right click on the Adapter icon and select Additional Functions, Export, DBM/nxd.

 

9.Select a name for the file, e.g.: adapter.nxd. Save the file in your current MATLAB workspace.

 

Note: this operation will generate two files:

 

adapter.nxd

adapter_nwid.nxd

 

10.In MATLAB, locate the files and use the following script to load the configuration on to the target machine

 

IO756                 = sg_IO75X_struct();   % Define default structure

 

IO756.protocol        = 'IO756';             % Protocol Speedgoat ID

IO756.cardid          = '1';                 % Card ID

IO756.configfiles     = '2';                 % No. of configuration files

IO756.configfile1     = adapter.nxd';         % Configuration filename

IO756.configfile2     = adapter_nwid.nxd';   % Configuration filename

 

sg_IO75X_loadconfig(IO755);  

 

Note: repeat point 8 to 10 every time you modify the Adapter configuration in SYCON.net.

 

11.Configure the driver blocks according to the configuration defined in SYCON.net.

 

12.Once you build the model, firmware and configuration are loaded to the IO756 EtherNet/IP Adapter I/O module.

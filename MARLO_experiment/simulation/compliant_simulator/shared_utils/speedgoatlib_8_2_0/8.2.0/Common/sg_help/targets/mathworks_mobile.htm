Configure MATLAB to always run as administrator

 

Windows 7 and Windows Vista

 

1.Right click the MATLAB shortcut on your desktop, taskbar, or start menu.
2.Click on "Properties".
3.Navigate to "Compatibility".
4.Check the box "Run this program as an administrator".
5.Click OK to save changes.

 

Windows 8

 

1.In File Explorer, navigate to the MATLAB executable file e.g. C:\Program Files\MATLAB\R2013b\bin\matlab.exe.
2.Right click on matlab.exe.
6.Click on "Properties".
7.Navigate to "Compatibility".
8.Check the box "Run this program as an administrator".
9.Click OK to save changes.

IO331 > Driver configuration > Quadrature > QAD

Input and output ports

Input and output ports

The width of all the inputs and all the outputs are equal to the channel count as defined in the channel vector parameter.

 

 

Input port

Description

Async. value

The value to load to the counter when an asynchronous load is requested.

 

This port is enabled/disabled using the 'Show asynchronous load value input port' parameter.

Async. command

Options for the input are:

 

•0: position value is not modified

•1: position value is set to zero

•2: Position value is reset to the value(s) defined in parameter field “Preload counter value”, or if the “asynchronous value” input port is present (by enabling the 'Show asynchronous load value input port' parameter), the position value is reset to the value from this port.

 

Note that using this port introduces additional latency.

This port is enabled/disabled using the 'Show asynchronous command register input port' parameter.

Load on index value

The quadrature decoder will reload this value to the counter value when index control mode is set to reload on index.

This port is enabled/disabled using the 'Show load on index input port' parameter.

 

 

Output port

Description

Current position

The most recently measured position.

 

This port is enabled/disabled using the 'Show current position output port' parameter.

Latched position

The position at the moment when the latch occurred.

 

This port is enabled/disabled using the 'Show latched position output port' parameter.

Direction

The direction of rotation for the most recently measure position.

 

Possible values for each vector element are:

 

•0: Counter-clockwise

•1: Clockwise

 

This port is enabled/disabled using the 'Show direction output port' parameter.

Index found

Indicates if the C/index position has been reached.

 

Possible values for each vector element are:

 

•0: Encoder hasn't reached C/Index position

•1: Encoder has reached C/Index position. The value remains at 1 until counting is reset for example via a Cmd input port command.

 

This port is enabled/disabled using the 'Show index found output port' parameter.

Latch

Indicates if new latched data is present.

 

If the index control mode is not used (ignore index input), the C/Index line can be used for the latched data, see parameter “Latch mode” for further information. If the C/Index I/O line is used, an additional D (dedicated latched) I/O line must be provided.

 

Possible values for each vector element are:

 

•0: No new position data (no new trigger event was asserted)

•1: New position data received (trigger event). This will clear after each sample hit.

 

This port is enabled/disabled using the 'Show latch status output port' parameter.

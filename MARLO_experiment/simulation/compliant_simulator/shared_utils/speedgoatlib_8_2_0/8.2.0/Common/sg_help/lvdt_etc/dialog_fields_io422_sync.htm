Tab: Module setup

 

 

Parameter

Description

IO422 module identifier

As it's possible to have more than one IO422 module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one IO422 module then leave the default value (1).

Slot

As the IO422 module can be equipped with up to three function specific NAII sub-modules, the slot in which this submodule is installed must be specified.

Channels vector

A vector of the channels to enable.

Channels in the range 1 to 4 can be specified.

Sample time [s]

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

PCI slot (-1: autosearch)

If only one module of this type is present in the target machine, leave as default ( '-1') to automatically locate the board. If more than one module of this type is present, enter the bus number and the PCI slot number of the board associated with this driver block using the format '[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

 

 

Tab: Synchro/Resolver

 

 

Output port

Description

Bandwidth vector

The bandwidth for each channel.

 

Channels can individually be programmed between 6 and 1280Hz with even numbers only.

 

A value of 0 causes the the sub-module to automatically set the bandwidth.

Ratio vector (1-256)

The desired ratio as an integer number corresponding to the pair of channels to be used for a two-speed or multiple-speed configuration.

 

The range is from 1 to 256, where 1 is for single speed.

 

Example: for a ration of 36:1 enter integer 36.

 

High resolution vector

The resolution of each channel can be individually configured.

 

This parameter should be a scalar or a vector with the same length as the 'Channel vector' parameter. Possible values are:

•0 to select 16-bit

•1 to select 24-bit

Synchro vector

The S/D measurement format of each channel can be individually configured.

 

This parameter should be a scalar or a vector with the same length as the 'Channel vector' parameter. Possible values are:

•0 to select Resolver

•1 to select Synchro

Encoder vector

This parameter controls the encoder or commutation outputs.

 

This parameter should be a scalar or a vector with the same length as the 'Channel vector' parameter. Values should be entered according to the following:

 

• Commutator — Enter values from the set [4 6 8].

 

These values specify the number of poles for a commutator output.

 

Max RPS vector

The maximum measurable velocity (in revolutions per second) for each channel.

 

This parameter should be a scalar or a vector with the same length as the 'Channel vector' parameter.

Latch all channels before reading position data

When checked all channels are guaranteed to be latched before being read.

 

This ensures that all channels are sampled at exactly the same time. Note that selecting this option incurs a small additional resource overhead.

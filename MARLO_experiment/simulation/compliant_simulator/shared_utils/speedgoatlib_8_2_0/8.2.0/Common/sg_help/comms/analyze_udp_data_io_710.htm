Analyze UDP data transmission

Save, build and run the model on the target machine.

 

To confirm whether the data is being transmitted, connect an Ethernet cable from the  IO70X/IO71X I/O module to the host (you can use the same used to load the model).

 

Use Wireshark or any other network analyzer on your host computer to capture the broadcast message.

The captured message must look exactly like this:

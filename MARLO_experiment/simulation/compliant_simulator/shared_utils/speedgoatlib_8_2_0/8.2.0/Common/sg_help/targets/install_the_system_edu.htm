Target machine installation

Place your Education real-time target machine in a stable environment.

 

 

Monitor

 

You can connect one DVI monitor to your target machine. Note that all other available monitor connectors are disabled, but you can use a VGA monitor with a DVI-to-VGA adapter. The monitor is not mandatory, but recommended for kernel transfer and debugging operations.

 

For the first session with your new target machine, please connect a monitor. The target main screen provides helpful information:

•Simulink Real-Time version
•Number of CPU cores loaded
•Available memory
•TCP/IP configuration for host-target communication

 

You can also use Target Scopes in the Simulink model to visualize signals and parameters at execution time.

 

If you don't use a monitor during real-time operations, you could run the kernel in text mode, to improve performance.

 

If you do use a monitor, connect it before powering up the target machine. Make sure that the monitor is well connected and do not unplug it during model execution. This could have an impact on the application stability and in some particular cases, crash the system. If this problem occurs, you must restart the target machine.

 

Keyboard

 

You can connect an USB keyboard to your target machine. Please use one of the four USB ports available on the side. The USB port under the top cover is reserved for kernel transfer.

 

With the USB keyboard you can enter the BIOS and use Simulink Real-Time commands. It is not possible to use DOS commands with Education real-time target machine.

 

The keyboard is not mandatory for Simulink Real-Time.

 

To activate the command line interface once you have a target application running please type C at the attached keyboard. For a list of commands available in the Simulink Real-Time environment, please refer to the Simulink Real-Time documentation.

 

Power supply

 

The Education real-time target machine uses different input voltage range, depending on the ordering option. Please refer to the Technical Reference Manual, which contains specific information for your target machine.

 

An AC/DC converter is provided with your system. Connect this external power supply to the target machine power connector. If you use your custom external power supply please make sure that your target machine supports its voltage.

 

Once the power supply is connected, the system automatically powers up.

 

The system usually takes a few seconds to load the Simulink Real-Time kernel stored in the target machine main drive. Once the kernel is loaded, you will see some kernel information on the target screen.

 

Power-on / Power-off

 

The system powers on as soon as the power supply is connected. To switch off the system, remove the power supply.

 

Alternatively you can use the power button which is located under the top cover.

IO106-32/16 variant with 80-pin P50E80 connector (Item ID 2A106)

This simultaneous sampling I/O module implements either 32 single ended (31 in pseudo differential mode) or 16 differential analog input channels. Each channel has a dedicated A/D converter resulting in very low sampling and conversion times.

 

IO106-64/32 variant with 80-pin P50E80 connector (Item ID 2B106)

This simultaneous sampling I/O module implements either 64 single ended (63 in pseudo differential mode) or 32 differential analog input channels. Each channel has a dedicated A/D converter resulting in very low sampling and conversion times.

 

IO106-32/16 variant with 68-pin MDR connector (Item ID 2C106)

As the 2A106 variant, but with an MDR connector.

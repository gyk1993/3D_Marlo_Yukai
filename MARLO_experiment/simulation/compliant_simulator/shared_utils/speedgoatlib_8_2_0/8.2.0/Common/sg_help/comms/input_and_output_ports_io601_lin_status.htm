IO601 > Driver configuration > LIN > LIN status driver block

Input and output ports

Input and output ports

Output port

Description

role

This port can have the following values:

•0: the board acts as a slave
•1: the board acts as a master

bus load

The current percentage (0..100) of LIN bus load.

baudrate

The currently configured baudrate.

error flag

This port can have the following values:

•0: no error
•1: data overrun
•2: LIN controller is still in init mode

SYCON.net Configuration tool

SYCON.net Configuration tool

The SYCON.net configuration tool is used to configure the network.

 

1.Open SYCON.net and locate the “CIFX RE/EIM” EtherCAT Master. Drag and drop the device to the main panel.

 

 

2.From the list of Stand-Alone Slave and Slave you can select devices to add to the EtherNet/IP network. Simply drag and drop the devices.

 

 

Note: you can add EDS files of others devices by following Network, Import Device Description.

 

3.Double-click on the icon to open the Configuration window.

 

4.Navigate to Configuration, Network Settings. You can select all available options.

 

 

5.Navigate to Configuration, Scanlist. For each connected device you can modify the IP address, the Requested Packet Interval (RPI, in ms), and the timeout.

 

6.Navigate to Configuration, Address Table. Note the Length and Address information that you will need when configuring the Simulink driver blocks.

 

 

7.Navigate to Configuration, Scanner Settings.

 

 

8.Apply the configuration and close the windows. Right click on the Scanner icon and select Additional Functions, Export, DBM/nxd.

 

9.Select a name for the file, e.g.: scanner.nxd. Save the file in your current MATLAB workspace.

 

Note: this operation will generate two files:

 

 scanner.nxd

 scanner_nwid.nxd

 

10.In MATLAB, locate the files and use the following script to load the configuration to the target machine

 

IO755                 = sg_IO75X_struct();   % Define default structure

 

IO755.protocol        = 'IO755';             % Protocol Speedgoat ID

IO755.cardid          = '1';                 % Card ID

IO755.configfiles     = '2';                 % No. of configuration files

IO755.configfile1     = scanner.nxd';         % Configuration filename

IO755.configfile2     = scanner_nwid.nxd';   % Configuration filename

 

sg_IO75X_loadconfig(IO755);  

 

Note: repeat point 8 to 10 every time you modify the Scanner configuration in SYCON.net.

 

11.Configure the driver blocks according to the configuration defined in SYCON.net.

 

12.Once you build the model, firmware and configuration are loaded to the IO755 EtherNet/IP Scanner I/O module.

 

13.The SYCON.net configuration tool is used to configure the network.

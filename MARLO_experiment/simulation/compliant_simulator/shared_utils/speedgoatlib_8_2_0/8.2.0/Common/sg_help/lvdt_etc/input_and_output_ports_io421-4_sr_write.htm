Input and output ports

Input and output ports

Input port

Description

rotation rate

The required rotational speed.

 

Units are RPS.

 

This port is only enabled when the 'Rotation mode' parameter is set to Speed driven or Start and stop.

start angle

The required angle at the beginning of the sequence.

 

Units are Radians.

 

This port is only enabled when the 'Rotation mode' parameter is set to Start and stop.

stop angle

The required angle at the beginning of the sequence.

 

Units are Radians.

 

This port is only enabled when the 'Rotation mode' parameter is set to Start and stop.

angle

The required angle.

 

Units are Radians.

 

This port is only enabled when the 'Rotation mode' parameter is set to Angle driven.

 

All ports are vectors. The width of the vector, and the order of the corresponding channels, is dependent on the configuration of the 'Channel vector' parameter.

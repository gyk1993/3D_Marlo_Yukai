IO205 > Driver configuration > Write driver block

Dialog fields

Dialog fields

 

Parameter

Description

IO205 module identifier

As it's possible to have more than one IO205 module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one IO205 module then leave the default value (1).

Port type

Select one of the five available port types (see 'Inputs and output ports').

Sample time

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

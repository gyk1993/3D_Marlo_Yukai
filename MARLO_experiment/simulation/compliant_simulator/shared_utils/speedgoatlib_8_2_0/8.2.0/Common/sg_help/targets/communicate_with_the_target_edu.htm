Communicate with the target

Once Windows network settings, MATLAB network and kernel settings, and kernel configuration loaded in the target machine correspond with each other, you are ready to communicate with host and target systems.

 

In MATLAB type xpctargetping to test the communication. If the command return success, all settings are correctly configured.

 

As the target machine uses a standard TCP/IP network configuration, you can also ping the target machine's IP address from the Windows Command prompt, e.g.: ping 192.168.7.1.

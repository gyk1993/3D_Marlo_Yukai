Actions

One of four possible actions can be triggered on each event (see Events section).

 

To select which event is is triggered the A-action and B-action parameters are configured with the following 'action values':

 

Action value

Description

0

No state change (0 stays 0, 1 stays 1)

1

Clear state (0)

2

Set state (1)

3

Toggle state (0 to 1, or 1 to 0)

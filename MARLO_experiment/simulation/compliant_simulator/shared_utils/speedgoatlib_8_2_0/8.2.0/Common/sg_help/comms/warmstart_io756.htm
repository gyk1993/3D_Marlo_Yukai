Warmstart

The warmstart mode allows a configuration subset to be automatically loaded to the Simulink driver block. This configuration is limited to the number of input and output bytes.

 

The required firmware files are normally installed on the main drive of your target machine. If for some reason the files aren't installed (e.g. the IO756 module was installed by the user), the files can be loaded on to the target machine using the following MATLAB commands:

 

IO756                 = sg_IO75X_struct();   % Define default structure

 

IO756.protocol        = 'IO756';             % Protocol Speedgoat ID

IO756.cardid          = '1';                 % Card ID

sg_IO75X_loadconfig(IO756);                   % Load configuration

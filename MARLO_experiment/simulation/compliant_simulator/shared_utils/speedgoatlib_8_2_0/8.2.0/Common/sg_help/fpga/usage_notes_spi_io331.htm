Usage notes

The SPI module can act as SPI Master or SPI slave.

 

Here is an example timing diagram of the signal generated / read by the SPI Master / Slave module:

 

 

 

Description:

T1        =        Start delay (master only)
T2        =        Clock divider

T3        =        Inter-word delay (master only)

T4        =        Stop delay (master only)

 

 

The SPI frame consists of a specific number of words (frame size), with each word consisting of a specific number of bits (word size). The total payload of the SPI frame is equal to the frame size multiplied by the word size.

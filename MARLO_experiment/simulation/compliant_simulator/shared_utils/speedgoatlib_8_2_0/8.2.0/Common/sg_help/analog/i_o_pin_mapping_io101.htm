Important notes

 

Analog inputs are floating, if not connected. All unused analog inputs should be connected to ground.

 

In single ended mode, pin 52 (sense) must be used for analog input ground.

 

Ground for analog outputs and digital I/O lines are pins 10, 11, 16, 17, 18, 35, 46, 47, 48 and 49.

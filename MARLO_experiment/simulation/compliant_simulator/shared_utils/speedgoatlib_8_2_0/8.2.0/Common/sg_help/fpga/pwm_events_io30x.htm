Events

Events

A-action and B-action parameters take values which correspond to the following events, which occur as the counter runs it's cycle of counting up and down.

 

Four events are possible for the A channel:

 

Event (see event sub-section)

Description

Zero

The counter values is now equal to zero

Period-Compare

The counter value is now equal to the Period-Compare value.

 

This is the 'C' point in the configuration examples in the 'How to configure' section.

A-Up-compare

The counter value is now equal to A-Compare value and the counter is counting up.

 

This is the A point in the configuration examples in the 'How to configure' section.

A-down-compare

The counter value is now equal to A-Compare value and the counter is counting down.

 

This is the A' point in the configuration examples in the 'How to configure' section.

 

 

Four events are possible for the B channel:

 

Event (see event sub-section)

Description

Zero

The counter values is now equal to zero

Period-Compare

The counter value is now equal to the Period-Compare value.

 

This is the 'C' point in the configuration examples in the 'How to configure' section.

B-Up-compare

The counter value is now equal to B-Compare value and the counter is counting up.

 

This is the B point in the configuration examples in the 'How to configure' section.

B-down-compare

The counter value is now equal to B-Compare value and the counter is counting down.

 

This is the B' point in the configuration examples in the 'How to configure' section.

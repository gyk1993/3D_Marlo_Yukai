Tab: Module setup

 

 

 

Parameter

Description

IO421 / IO422 module identifier

As it's possible to have more than one IO421 / IO422 module in a target machine, this parameter is used by the other driver blocks to identify a specific module. Select a unique number (1..10) for each of your modules. If your target machine only contains one IO421 / IO422 module then leave the default value (1).

Channels vector

The channels to enable.

This can be a scalar or a vector where the elements are 1, 2 or 3.

The elements of this parameter can be mixed in any order. This order has a direct influence on the port configuration, e.g. with 'Channels vector' parameter set to  [2, 3, 1], the first element of the vector accesses channel 2, the second element accesses channel 3 and the third element accesses channel 1.

Sample time [s]

Defines the base sample time at which this driver block gets its sample hit. It can also be set to -1 for inherited sample time.

 

Units are seconds.

PCI slot (-1: autosearch)

If only one module of this type is present in the target machine, leave as default ( '-1') to automatically locate the board. If more than one module of this type is present, enter the bus number and the PCI slot number of the board associated with this driver block using the format '[BusNumber, SlotNumber]'. To determine the bus number and the PCI slot number, type 'getxpcpci' in the MATLAB command window

Tab: Excitation

 

 

Parameter

Description

Expected excitation voltage (RMS)

This parameter informs the IO421 / IO422 I/O module about the expected voltage (RH and RL inputs).

The value is given in VRMS. Therefore, if your sine wave has an amplitude of 10 you have to divide it by √2.

 

 

Tab: A/B output

 

 

Parameter

Description

VLL output (RMS) vector

This parameter configures the line to line output voltage.

The value is given in VRMS.

Phase offset vector

This parameter defines the phase offset between A/B output and the reference signal.

The parameter can be defined with an accuracy of 0.1°.

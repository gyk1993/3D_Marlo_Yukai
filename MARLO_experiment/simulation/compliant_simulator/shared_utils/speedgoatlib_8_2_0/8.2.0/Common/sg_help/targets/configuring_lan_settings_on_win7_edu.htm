Configuring LAN settings on the host PC using Windows 7

Configuring LAN settings on the host PC using Windows 7

To easily identify the correct Ethernet controller that you are going to use, proceed as follows:

 

1.At your host PC navigate to Start, Control Panel, Network and Internet, Network and Sharing Center
2.Connect your Ethernet cable between a free LAN port of your host PC (or use a USB to Ethernet adapter if you have no spare LAN port available) and the LAN port labeled “Host link” of your target machine.
3.Power on your target machine
4.At your host PC click on the correct Local Area Connection (if more than one is displayed).

 

Once you have identified the correct Ethernet controller, click on the adapter Properties:

 

1.Select "Internet Protocol Version 4 (TCP/IPv4)" and click Properties
2.Select "Use the following IP address" and configure settings accordingly to your network requirements. If you want to use the default Speedgoat settings, you can insert:

 

IP address

192.168.7.2

Subnet mask

255.255.255.0

 

The network is now ready, confirm with OK to close all windows.

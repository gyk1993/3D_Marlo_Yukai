Center-aligned

Here is a timing diagram for a center-aligned (symmetrical) signal:

 

 

The counter increments until the half-period (C) and then decrements to zero again.

Duty cycles for the line A and B signals in this example are centered at the half-period C and are therefore symmetric.

 

In order to define at which points on the counting cycle the polarity of the signal changes the four 'A-action' elements are configured as in the following table.

'A-action' is either set by the 'Initial A-action matrix' parameter, or by the 'A-action' input port.

 

Event (see Events sub-section)

Configuration value

Zero

0 - No state change

Period-Compare

0 - No state change

A-Up

2 - Set state

A-Down

1 - Clear state

 

So for this example the A-action matrix would be [0,0,2,1]'.

 

The timing of the A-Up and A-Down events is defined by the 'A-compare count' input port (initialized by the 'Initial A-compare start state vector' configuration parameter).

 

The B signal is similarly configured.

 

The timing of the C event (the point when the counter begins counting down to zero, half of the period T), is defined by the 'Period-compare Count' and initialized by the 'Initial period-compare start state vector' configuration parameter. See the 'Compare values' section for information on calculating the PWM output frequency.

xPC Target    

Before You Start

xPC Target uses a model for reflective memory (also known as shared memory) that includes Simulink blocks, for the shared memory drivers, and MATLAB structures for defining shared memory and node initialization partitions. The topics in this section are

  • Create Shared Memory Partitions -- Before you begin to use SCRAMNet GT shared memory blocks, define a partition structure that defines how you want to allocate shared memory.
  • Initialize Shared Memory Nodes -- Before you begin to use SCRAMNet GT shared memory blocks, define a node initialization structure that defines how shared memory is allocated (partitioned) and how the board is configured.

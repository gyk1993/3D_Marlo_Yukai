SCGT Init
Configures the SCRAMNet GT shared memory board. This is a required block for any model that uses shared memory with the SCRAMNet GT board. It configures important registers on the board acording to the user-defined node structure.

Driver Block Parameters

Before you begin to configure these block parameters, be sure that you have a predefined node initialization structure. See Initialize Shared Memory Nodes.

Each model that uses shared memory must have one SCGT Init block for every SCRAMNet GT board in the system.

Node struct -- Enter the name of the predefined node initialization structure.

PCI Slot (-1:autosearch) -- If only one board of this type is physically present in the target PC, enter -1.
